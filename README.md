# 远程输入法 - Remoboard

远程输入法 是一个输入法应用，可以实现使用电脑给手机打字，让手机打字更快。使用远程输入法时，手机端会显示一个网址，在电脑浏览器中访问这个网址，即可开始输入。浏览器中输入的文字发送到手机，从而实现电脑给手机打字。
